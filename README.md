# 🏆 Kabaddi Arena – Android App Development using GenAI

## 📱 Overview
Kabaddi Arena is an Android application developed using Generative AI concepts to create a modern and engaging platform for kabaddi enthusiasts. The project demonstrates Android app development, UI/UX design, and AI-assisted development workflow.

---

## 🚀 Features

- 🏟️ Interactive Kabaddi Arena interface
- 👥 Team and player management
- 📅 Match schedules and updates
- 🔍 User-friendly navigation
- 🤖 Generative AI assisted development
- 📱 Responsive Android UI

---

## 🛠️ Technologies Used

- Java / Kotlin
- Android Studio
- XML
- Firebase
- Git & GitHub
- Generative AI Tools

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/PrateekshaDR/Android-app-development-using-GenAI-Kabaddi-Arena.git
```

### Open in Android Studio

1. Open Android Studio
2. Select **Open Existing Project**
3. Choose the project folder
4. Sync Gradle files
5. Run the application

# Android-app-development-using-GenAI-Kabaddi-Arena
Kabaddi-Arena is an Android app built using GenAI to track Kabaddi player performance. It records raids, tackles, and match actions to generate stats, heatmaps, and performance reports. The app helps players and coaches analyze strengths, improve skills, and support talent recognition in local tournaments.
